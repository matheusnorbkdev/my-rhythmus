#  MyRhythmus

**MyRhythmus** é um aplicativo de mobilidade física desenvolvido com foco em atletas, com o objetivo de melhorar o desempenho e **prevenir lesões musculoesqueléticas** através de rotinas inteligentes de movimento e alongamento.

---

## Sobre o projeto

O MyRhythmus nasce da necessidade de integrar tecnologia e saúde no dia a dia de atletas e praticantes de atividades físicas. Muitas lesões poderiam ser evitadas com preparação adequada — e é exatamente isso que o app propõe.

Com uma abordagem prática e acessível, o aplicativo ajuda o usuário a manter uma rotina eficiente de mobilidade, respeitando seus limites e evolução.

---

 Objetivos

*  Prevenir lesões musculoesqueléticas
*  Melhorar a mobilidade e flexibilidade
*  Auxiliar na recuperação muscular
*  Criar rotinas personalizadas para atletas
*  Incentivar hábitos saudáveis

---

 Funcionalidades (em desenvolvimento)

*  Criação de rotinas de mobilidade
*  Timer para exercícios
*  Acompanhamento de evolução
*  Sugestões inteligentes com uso de IA
*  Lembretes de treino
*  Interface moderna e intuitiva

---
##  Gamificação

O MyRhythmus utiliza conceitos de gamificação para tornar a prática de mobilidade mais envolvente e motivadora.
Através de sistemas de pontos, níveis, desafios e conquistas, o usuário é incentivado a manter consistência nos exercícios, transformando o cuidado com o corpo em uma experiência dinâmica e interativa.
Essa abordagem aumenta a adesão às rotinas e contribui diretamente para a prevenção de lesões e melhoria do desempenho físico.

## 🚀 Como rodar o projeto

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/my-rhythmus.git

# Entre na pasta
cd my-rhythmus

# Instale as dependências
npm install

# Inicie o projeto
npx expo start
```
